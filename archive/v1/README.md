# Archive — v1

These are the **original (v1)** Pre-cost skill files, superseded by **v2** in the
repo root (`precost-skill-v2.zip`) and under [`skill/`](../../skill/).

They are kept here for reference only. Do not install these — use the current v2
bundle instead.

- `precost-skill-v1.zip` — the original downloadable bundle.
- `skill/` — the original unzipped skill files (main file named `precost.md`).

The main difference in v2: a corrected cost formula (a one-level class bump for
heavy inputs / expensive tools plus the model multiplier, instead of the old
multi-factor formula that could detach the number from the Light / Medium / Heavy
badge), a corrected seat-price note, and README tidy-ups.
